درخواست‌های تغییر قانون اساسی

تغییرات متنی مشخص در قانون اساسی ایران پیشنهاد دهید. پیاده‌سازی اصلاحات خاص را بررسی و همکاری کنید.

مشاهده درخواست‌ها ایجاد درخواست

درخواست‌های فعال

محتوای درخواست‌ها به صورت خودکار از GitHub API به‌روزرسانی می‌شود...

  • 🔄 محتوا از طریق GitHub Actions واکشی و ترجمه می‌شود
  • 📝 درخواست‌ها شامل: عنوان، وضعیت، تغییرات، بررسی‌ها، آرا
  • 🌐 درخواست‌های انگلیسی به فارسی ترجمه می‌شوند
  • 👥 نظرات بازبینان و بازخورد جامعه را مشاهده کنید
  • ✅ وضعیت تأیید و آمادگی ادغام را پیگیری کنید

چگونه مشارکت کنیم

  1. ایجاد درخواست - تغییرات مشخص متن قانون اساسی پیشنهاد دهید
  2. بررسی تغییرات - اصلاحات پیشنهادی را با دقت بررسی کنید
  3. افزودن نظر - بازخورد روی خطوط مشخص ارائه دهید
  4. رأی‌گیری - از پیشنهادها حمایت کنید یا تغییر درخواست کنید
  5. همکاری - با نویسندگان برای تصحیح اصلاحات کار کنید
  6. ادغام - به تغییرات تأیید شده برای رسیدن به آستانه ۷۵٪ کمک کنید
