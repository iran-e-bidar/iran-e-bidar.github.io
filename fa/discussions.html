بحث‌های جاری

محتوای بحث‌ها به صورت خودکار از GitHub API به‌روزرسانی می‌شود...

  • محتوا از طریق GitHub Actions واکشی و ترجمه می‌شود
  • بحث‌ها شامل: عنوان، نویسنده، تاریخ، تعداد پاسخ
  • بحث‌های انگلیسی به فارسی ترجمه می‌شوند
  • برای مشاهده جزئیات روی هر بحث کلیک کنید

چگونه مشارکت کنیم

  1. شروع بحث - روی "شروع بحث جدید" کلیک کنید
  2. انتخاب دسته - موضوع مناسب قانون اساسی را انتخاب کنید
  3. به هر زبان بنویسید - از انگلیسی یا فارسی استفاده کنید، محتوا ترجمه می‌شود
  4. مشارکت کنید - به بحث‌های موجود پاسخ دهید و اجماع بسازید
