مسائل قانون اساسی

اصلاحات و بهبودهای مشخص قانون اساسی را پیگیری کنید. مسائل جاری را مشاهده کنید و راه‌حل‌های جدید پیشنهاد دهید.

ایجاد مسئله جدید مشاهده در گیت‌هاب

مسائل جاری

محتوای مسائل به صورت خودکار از GitHub API به‌روزرسانی می‌شود...

  • محتوا از طریق GitHub Actions واکشی و ترجمه می‌شود
  • مسائل شامل: عنوان، وضعیت، برچسب‌ها، مسئول، تاریخ‌ها
  • مسائل انگلیسی به فارسی ترجمه می‌شوند
  • فیلتر با برچسب‌ها: اصلاح، بهبود، اشکال، مستندات
  • پیشرفت و آرای جامعه را پیگیری کنید

چگونه مشارکت کنیم

  1. ایجاد مسئله - روی "ایجاد مسئله جدید" کلیک کنید
  2. انتخاب قالب - اصلاح، بهبود یا گزارش اشکال انتخاب کنید
  3. شرح مسئله - مسئله قانون اساسی را به وضوح توضیح دهید
  4. پیشنهاد راه‌حل - تغییرات یا بهبودهای مشخص پیشنهاد دهید
  5. افزودن برچسب - با مواد مربوط قانون اساسی برچسب‌گذاری کنید
  6. پیگیری پیشرفت - آرا و بازخورد جامعه را رصد کنید
