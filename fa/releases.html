انتشارات قانون اساسی

نسخه‌های منتشر شده اصلاحات تأیید شده قانون اساسی. انتشارات رسمی و نقاط عطف پیاده‌سازی را پیگیری کنید.

انتشارات منتشر شده

محتوای انتشارات به صورت خودکار از GitHub API به‌روزرسانی می‌شود...

  • 🔄 محتوا از طریق GitHub Actions واکشی و ترجمه می‌شود
  • 📦 انتشارات شامل: نسخه، تاریخ، تغییرات، لینک‌های دانلود
  • 🌐 انتشارات انگلیسی به فارسی ترجمه می‌شوند
  • 📄 به اسناد رسمی اصلاحات دسترسی پیدا کنید
  • 🎯 پیشرفت پیاده‌سازی و پذیرش را پیگیری کنید

چگونه مشارکت کنیم

  1. بررسی انتشارات - اصلاحات منتشر شده قانون اساسی را بررسی کنید
  2. دانلود اسناد - به متون رسمی اصلاحات دسترسی پیدا کنید
  3. ارائه بازخورد - در مورد اثربخشی پیاده‌سازی نظر دهید
  4. ایجاد انتشار - اصلاحات تأیید شده را برای انتشار بسته‌بندی کنید
  5. مستندسازی تغییرات - یادداشت‌های انتشار و خلاصه‌های واضح بنویسید
  6. ترویج پذیرش - به گسترش آگاهی از اصلاحات جدید کمک کنید
