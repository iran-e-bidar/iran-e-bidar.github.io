پروژه‌های قانون اساسی

پیشرفت ابتکارات قانون اساسی و بسته‌های اصلاحی را رصد کنید. تلاش‌های هماهنگ در چندین ماده را پیگیری کنید.

مشاهده پروژه‌ها ایجاد پروژه

پروژه‌های فعال

محتوای پروژه‌ها به صورت خودکار از GitHub API به‌روزرسانی می‌شود...

  • محتوا از طریق GitHub Actions واکشی و ترجمه می‌شود
  • پروژه‌ها شامل: نام، وضعیت، پیشرفت، جدول زمانی
  • پروژه‌های انگلیسی به فارسی ترجمه می‌شوند
  • برد پروژه‌ها را با سازماندهی کارها مشاهده کنید
  • نقاط عطف و درصد تکمیل را پیگیری کنید

چگونه مشارکت کنیم

  1. پیوستن به پروژه - روی هر پروژه فعال کلیک کنید تا شرکت کنید
  2. ایجاد پروژه - ابتکار جدید قانون اساسی شروع کنید
  3. افزودن کارها - اصلاحات را به گام‌های قابل مدیریت تقسیم کنید
  4. به‌روزرسانی پیشرفت - کارها را در مراحل گردش کار جابجا کنید
  5. هماهنگی - با اعضای تیم روی تغییرات پیچیده کار کنید
  6. بررسی - در بررسی نقاط عطف پروژه شرکت کنید
