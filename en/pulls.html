How to Contribute

  1. Create PR - Propose specific constitutional text changes
  2. Review Changes - Examine proposed amendments carefully
  3. Add Comments - Provide feedback on specific lines
  4. Vote - Support or request changes to proposals
  5. Collaborate - Work with authors to refine amendments
  6. Merge - Help approved changes reach 75% threshold
